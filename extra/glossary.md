*[сетевых представлений]: Онлайн-комнаты, где один пользователь (ведущий) что-то показывает, а остальные (участники) смотрят и периодически посылают обратную связь.

*[карточки сценария]: Карточка, отображаемая для каждого сценария на главной странице проекта.

*[PartyFlow]: Граф состояния игры, содержащий данные для организации взаимодействия пользователей онлайн-комнаты.

*[PartyQuery]: Узел графа состояния игры, содержащий данные о текущем экране для ведущего и участников.

*[WebPartySpec]: Спецификация формата сценариев WebPartyTime для языка разметки TOML.

*[ведущего]: Организатор онлайн-комнаты, на экране которого отображается общая для всех информация.
