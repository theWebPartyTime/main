# WebPartyTime
 Открытая система для проведения интерактивных презентаций прямо в браузере.


>Чтобы узнать более подробную информацию о проекте, обратитесь к его 
[официальной онлайн-документации](https://thewebpartytime.github.io/main/).


## Быстрый старт
Чтобы запустить проект локально и увидеть его в действии:


```shell
git clone --recursive git@github.com:theWebPartyTime/main.git
```

```shell
docker-compose -f podman-compose-prod.yml up --build 
```

Запущенное приложение будет доступно по адресу http://127.0.0.1:5173
