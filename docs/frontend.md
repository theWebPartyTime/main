# Frontend

## Роутинг
`/` - Страница онбординга (если пользователь не выбрал продолжить бег регистрации или не создал аккаунт) либо основная страница, где можно создавать комнаты и подключаться к ним


`/game` - Отображение идущей игры (и для ведущего и для игроков)


`/design` - Редактор сценариев
