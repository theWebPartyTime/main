# Основы
## Что это?
**WebPartyTime** - платформа для создания и проведения интерактивных представлений. 
> Презентации с голосованием или разделенным переключением слайдов, игры-викторины, текстовые онлайн-игры и генераторы для идей - всё это можно сделать средствами WebPartyTime.

## PartyFlow
**Цель проекта** - предоставить удобный и гибкий набор инструментов для создания многопользовательского опыта в формате "ведущий-участники".


Для её достижения проектом предлагается решение PartyFlow - идея представить каждое взаимодействие в виде узла общего графа. Таким образом, каждый узел PartyQuery содержит информацию о содержании интерфейса ведущего и игроков и условия перехода к другим узлам.
