## ==Что это?==
Если вкратце, то WebPartyTime - ==платформа для создания и проведения сетевых представлений==.

!!! note ""
    Презентации с голосованием или разделенным переключением слайдов, 
    игры-викторины, генераторы идей для продуктовых команд - всё это можно сделать средствами WebPartyTime.

==Цель проекта== - предоставить удобный и гибкий набор инструментов для создания многопользовательского опыта в формате "ведущий-участники".


При этом ==платформа независима от внешних сервисов== и при необходимости может изменяться и запускаться полностью локально. 

## ==Функции==
* Проведение ==интерактивных многопользовательских презентаций== прямо в браузере
* ==Гибкий редактор сценариев== для разработки собственных форматов презентаций
* ==Модульная архитектура== для высокой расширяемости проекта разработчиками
